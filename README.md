# **Website Admin Penjualan (Noctics)**
Website ini berisi landing page dan admin panel. Pada pengembangannya berfokus pada backend dan integrasi database ke website.

**Tujuan Proyek:**
- Memudahkan admin untuk menambahkan data baru (inventori dan member)
- Memudahkan pengelolaan data inventori dengan menerapkan konsep CRUD

## **Fitur Utama**
**User:**
- Akses landing page

**Admin Panel:**
- Login admin
- CRUD (Create, Read, Update, Delete) data
- Pengelolaan data inventori dan member
- Manajemen data yang tersimpan di database SQL

**Alur Kerja Aplikasi:**
1. Admin mengakses melalui landing page
2. Login sebagai admin untuk masuk ke admin panel

**Melalui proyek ini, saya mempelajari:**
- Implementasi CRUD sederhana menggunakan PHP dan SQL
- Pembuatan admin panel untuk manajemen data
- Sinkronisasi frontend (admin panel) dan backend

**Status Proyek:**
Proyek ini telah selesai dan dapat dikembangkan lebih lanjut.

**Cara Mengakses:**
- Ekspor data ke sql (noctics)
- Simpan proyek di xampp/htdocs/noctics/
- Akses di http://localhost/noctics/index.html
- Login sebagai admin: admin
